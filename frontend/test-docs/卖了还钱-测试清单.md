## 测试环境

- 后端: `cd backend && python app.py`
- 前端: `cd frontend && npx expo start`
- 前置: 衣柜中有标记为"不想要"的衣物

## 功能测试

### 1. 列表展示

| # | 测试项 | 预期 | 结果 |
|---|--------|------|------|
| 1.1 | 有"不想要"衣物 | 展示待转卖列表，每张卡片含图片/名称/分类/购入价 | ⬜ |
| 1.2 | 无"不想要"衣物 | 显示空状态引导："暂无待转卖物品" | ⬜ |
| 1.3 | 状态徽章 | 待售=黄色 议价中=蓝色 已售出=绿色 | ⬜ |
| 1.4 | 已售出物品 | 卡片显示半透明+已售徽章，无操作按钮 | ⬜ |

### 2. 统计条

| # | 测试项 | 预期 | 结果 |
|---|--------|------|------|
| 2.1 | 预期回血 | = 所有待售已标价物品的标价总和 | ⬜ |
| 2.2 | 已回血 | = 所有已售出物品的售价总和 | ⬜ |
| 2.3 | 待转卖件数 | = 非已售出的不想要物品数量 | ⬜ |
| 2.4 | 已售件数 | = 已售出物品数量 | ⬜ |
| 2.5 | 标价后刷新 | 预期回血数字更新 | ⬜ |

### 3. 标价功能

| # | 测试项 | 预期 | 结果 |
|---|--------|------|------|
| 3.1 | 点"标价" | 弹出 Modal，显示物品名和购入信息 | ⬜ |
| 3.2 | 输入价格 | 数字键盘，输入正常 | ⬜ |
| 3.3 | 点"确定" | Modal关闭，卡片显示挂售价，notes含RESELL数据 | ⬜ |
| 3.4 | 点"取消" | Modal关闭，价格不变 | ⬜ |
| 3.5 | 已标价再点"改价" | Modal预填当前价格 | ⬜ |
| 3.6 | 输入非数字 | 转为0或不处理 | ⬜ |

### 4. 确认售出

| # | 测试项 | 预期 | 结果 |
