# 穿搭日历 — 开发计划

## 功能概述

月度穿搭日历视图，展示每日穿搭记录、物品CPW（单次穿着成本）、月度排行。

## 数据基础（已就绪）

- `GET /api/outfits?start_date=&end_date=` — 日期范围查询
- `GET /api/outfits/date/<date>` — 单日查询
- `DELETE /api/outfits/<id>` — 删除穿搭
- 物品含 `purchase_amount` + `wear_count`，CPW = price / wear_count

## 组件

| `src/components/MonthCalendar.jsx` | 月历网格组件 |

## 状态

```
currentDate: Date           // 当前查看的年月
selectedDate: string|null   // 选中日期 YYYY-MM-DD
outfits: Map                // dateKey → outfit
loading: boolean
```

## 交互

1. 默认当月 → GET outfits 范围查询 → 标记有穿搭的日期
2. 点有穿搭日期 → 展开详情卡（物品/CPW/备注）+ 删除按钮
3. 点空日期 / 点 [+穿搭] → 跳转 ootd-lab（后续传参联动）
4. 左右箭头切换月份 → 重新查询
5. 底部本月 CPW 排行
