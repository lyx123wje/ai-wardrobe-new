# 穿搭日历 — 测试清单

> 状态: 🟡 待测试
> 创建: 2026-05-10

## 测试环境

- 后端: `cd backend && python app.py`
- 前端: `cd frontend && npx expo start`
- 前置: 确保衣柜中有衣物数据，且有穿搭记录（可在穿搭实验室创建）
