## 思想演化阶段

```
早年 (21-31):  朱学信徒 → 格竹失败 → 佛老探索
中年 (35-37):  政治打击 → 龙场悟道 → 心即理
中年后期 (38-49): 知行合一 → 事功实践 → 讲学传播
晚年 (50-57):  致良知 → 天泉证道 → 四句教
```

### 关键转折点
1. **格竹失败（21岁）** — 对朱学产生根本怀疑
2. **龙场悟道（37岁）** — 从外求转向内求
3. **平宁王之乱后（50岁）** — 致良知成熟
