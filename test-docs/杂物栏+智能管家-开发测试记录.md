## 概览

在衣柜页基础上新增杂物栏（独立 CRUD + 分类Tab）和衣柜智能管家（DeepSeek 对话 + 自动操作）。

涉及文件：

| 文件 | 操作 |
|------|------|
| `clothes-backend/database.py` | 修改：新建 `misc_items` 表、补齐 `wardrobe_items.notes` 字段、新增 misc CRUD |
| `clothes-backend/app.py` | 修改：新增 `/api/misc` 4 个端点、新增 `/api/wardrobe/ask` 端点、修复 notes 传参 |
| `clothes-backend/mind_engine.py` | 修改：新增 `ask_wardrobe_butler` / `build_wardrobe_butler_prompt` / `call_llm_long` |
| `src/utils/constants.js` | 修改：CATEGORIES 新增 `'杂物'`，颜色 `#8B7355` |
| `src/api/misc.js` | 新建：`fetchMiscItems / createMiscItem / updateMiscItem / deleteMiscItem` |
| `src/api/wardrobe.js` | 修改：新增 `askWardrobe(question)` |
| `src/components/MiscItemCard.jsx` | 新建：杂物网格卡片（缩略图+名称+位置标签） |
| `src/components/MiscAddModal.jsx` | 新建：杂物添加弹窗（拍照/选图→填名称/位置/备注→保存） |
| `src/components/ButlerChat.jsx` | 新建：管家对话 Modal（AI气泡+操作卡片+快捷提问） |
| `app/wardrobe.jsx` | 重写：双FAB、杂物分类切换、🤖管家按钮、杂物详情Modal |
| `app/index.jsx` | 修改：主页背景换成 `4.png` |

## 备注

- 杂物与衣物数据完全隔离（不同表、不同API）
- 管家 actions 由后端自动执行（非前端手动调），执行后前端刷新数据
- 管家依赖 DeepSeek API Key（`.env` 中配置），如 Key 失效则返回"AI 服务不可用"
- 首次启动后端会自动 ALTER TABLE 补齐 `wardrobe_items.notes` 列（兼容旧数据库）
